User Guide

Install and use Zync Wallet

Overview

Zync Wallet supports both Zinc Protocol (ZRC-20 tokens, NFT collections via OP_RETURN) and Zerdinals Protocol (witness inscriptions).

Note: Currently not on Chrome Web Store. Install manually using Developer Mode.

Installation

1

Download Extension

Click the button below to download zync-wallet.zip. Save it somewhere easy to find (Desktop or Downloads folder).

Download Now
2

Extract the ZIP File

Windows: Right-click the ZIP file → Select "Extract All" → Click "Extract".
Mac: Double-click the ZIP file to automatically extract it.
After extracting, you should have a folder containing files like manifest.json, background.js, popup.js, and an icons folder.

3

Open Browser Extensions Page

Open your browser (Chrome, Brave, or Edge). In the address bar, type chrome://extensions and press Enter.
Alternative: Click the three-dot menu (top right) → More Tools → Extensions.

4

Enable Developer Mode

On the extensions page, look for "Developer mode" toggle in the top-right corner. Click it to turn it ON - it should turn blue/highlighted.

5

Load the Extension

After enabling Developer Mode, you'll see new buttons appear. Click "Load unpacked".
A file browser will open - navigate to the folder you extracted in Step 2 (the one containing manifest.json). Select that folder and click "Select Folder".
Done! Zync Wallet icon should now appear in your browser toolbar.

First Time Setup

Create New Wallet

  • Click Create New Wallet
  • Write down your 12-word seed phrase
  • Set a strong password

Import Wallet

  • Click Import Wallet
  • Enter seed phrase or private key
  • Set a password

Using Both Protocols

Zinc Protocol

Create ZRC-20 tokens and NFT collections using OP_RETURN inscriptions.

Zerdinals Protocol

Inscribe data and images using scriptSig in transaction inputs.

Security

  • Never share your seed phrase
  • Write it down offline
  • Use a strong password
  • Lock wallet when not in use
  • Verify addresses before sending

Troubleshooting & FAQ

Click any question to see the answer

Extension not appearing after installation?

+

Make sure Developer Mode is enabled and you selected the correct folder (the one containing manifest.json). Try refreshing your extensions page or restarting your browser.

Can't see my transactions?

+

Ensure you're connected to the correct network (mainnet/testnet) in settings. Your wallet needs to sync with the blockchain, which may take a few moments.

Lost access to my wallet?

+

You can recover your wallet using your 24-word seed phrase. Click "Import Wallet" and enter your seed phrase. Your funds are always recoverable as long as you have your seed phrase.

How do I switch between mainnet and testnet?

+

Go to Settings → Network and select your preferred network. This is useful for testing before deploying to mainnet.

Transactions failing or stuck?

+

Check that you have sufficient ZEC for both the transaction amount and network fees. If stuck, the transaction may need more time to confirm on the blockchain.

Can I use Zync Wallet on mobile?

+

Currently Zync Wallet is available as a browser extension for desktop Chromium-based browsers only. Mobile support may come in future updates.

Where are my inscriptions shown?

+

NFTs and ZRC-20 tokens appear in the "Inscriptions" tab. Make sure you're connected to the correct network and your inscriptions have been indexed.

How do I backup my wallet?

+

Write down your 24-word seed phrase and store it in a secure offline location. Never store it digitally or share it with anyone. This is the only way to recover your wallet.
