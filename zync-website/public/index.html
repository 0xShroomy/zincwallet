Frequently Asked Questions

Click any question to see the answer

What is Zync Wallet?

+

Zync Wallet is the first browser extension wallet that supports both Zinc and Zerdinals protocols on Zcash. It's a non-custodial wallet that lets you manage ZEC, ZRC-20 tokens, and NFT inscriptions all in one place.

What's the difference between Zinc and Zerdinals?

+

Zinc Protocol uses OP_RETURN inscriptions for ZRC-20 tokens and NFT collections, while Zerdinals uses scriptSig in transaction inputs to inscribe data. Zync Wallet supports both protocols seamlessly.

Is Zync Wallet safe?

+

Yes! Zync Wallet is non-custodial, meaning you control your private keys. Your seed phrase and private keys never leave your device. Always backup your seed phrase and never share it with anyone.

Which browsers are supported?

+

Zync Wallet works on all Chromium-based browsers including Google Chrome, Brave, Microsoft Edge, and Opera. Simply load the unpacked extension in developer mode.

Does it support testnet?

+

Yes, Zync Wallet supports both mainnet and testnet. You can switch networks in the settings to test your dApps or inscriptions before going live.

Can I recover my wallet if I lose access?

+

Yes, as long as you have your 24-word seed phrase. You can import your wallet on any device using your seed phrase. This is why it's critical to write it down and store it securely offline.

How do I create ZRC-20 tokens or NFTs?

+

Zync Wallet provides built-in tools for minting ZRC-20 tokens and creating NFT collections using the Zinc Protocol. Check our documentation for step-by-step guides.

Are there any fees?

+

Zync Wallet is completely free to use. You only pay standard Zcash network transaction fees. No hidden fees, no subscriptions.

Can developers integrate Zync Wallet?

+

Absolutely! Zync Wallet provides a JavaScript API for dApp developers. Check our Developer Guide for integration examples and TypeScript definitions.
