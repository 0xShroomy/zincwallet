Easy Installation

1

Download Extension

Download the latest zync-wallet.zip file from the button below.

2

Extract Files

Unzip the downloaded file to a location on your computer.

3

Enable Developer Mode

Open Chrome, go to Extensions, and enable Developer mode (top right toggle).

4

Load Extension

Click Load unpacked and select the extracted dist folder.

5

Start Using Zync

Click the extension icon, create or import a wallet, and start managing your Zcash assets.
